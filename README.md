# MIPS Assembly Simulator (C++)

A simple **MIPS Assembly Simulator** implemented in C++.  
The simulator reads `.asm` files, interprets instructions, and displays the resulting register values.

---

## Project Features

- Reads and executes **MIPS assembly (.asm) files**
- Simulates basic **MIPS register operations**
- Displays **register values after execution**
- Supports the following instructions:
  - `ADD`
  - `SUB`
  - `ADDI`
- Command-line based execution
